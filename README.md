## hexo-deployer-oss ![NPM version](https://img.shields.io/npm/v/hexo-deployer-oss.svg?style=flat)

Aliyun-OSS deployer plugin for Hexo.

### Installation
```bash
$ npm install hexo-deployer-oss
```

### Example
```js
var hexoDeployerOss = require('hexo-deployer-oss');
```

### API
check this file: `index.js`
